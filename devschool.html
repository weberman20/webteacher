Здесь вы сможете обучиться базовым знаниям в сфере веб разработки.


1)Для начала вам потребуеться установить текстовый редактор для написания кода. Вы можете скачать любой редактор для написания, но для написания хороших и качественных сайтов вам потребуются профиссиональные текстовые редакторы вы можете выбрать подходящий для вас редактор которые вы сможете выбрать с написанных мной удобный редакторов с низу.

  • 1.Sublime Text 3
  • 2.Atom
  • 3.Py Charm
  • 4.VS Code
  • 5.Brackets

Если вы только начинающий веб програмист то вы можете пройти бесплатные курсы в ютубе по HTML.

2)Если вы уже прошли курс по HTML вы можете переходить к обучению CSS но, не забывайте практиковать HTML так как он пригодиться во всех ваших работах и не стоит его не изучить до конца. Если вы хотите уже практиковаться то вы можете поработать над маленьким и не сложным проэктом чтобы хорошо запомнить то что вы учили на курсе HTML. Для написания кода CSS вам пригодиться создать текстовый документ с расширением .css пример: (style.css) после того как вы создали css документ вы можете закоментировать ваши знания для того случая если вы забудете что-то чтобы вы смогли от туда посмотреть и решить вашу проблему. Чтобы сделать коментарий в css вам надо ввести /**/ и написать между ними ваш коментарий.

Если вы захотели приступить к курсам CSS вы можете это сделать нажать кнопку с низу.

После того как вы прошли Курс по CSS практикуйте их пока не добьетесь успеха и создадите свой первый полноценный сайт. Удачи вам в Своих целях.
